Название нашего проекта -  "CutFrame"

тема - онлайн курсы

описание нашего проекта:

наш проект - онлайн курсы по видеомонтажу (эдитингу) которые помогают начинающим, или уже тем кто имеют опыт подтянуть свои знания, узнать чтото новое. Мы предлагаем обучение в видеоредакторах таких как CapCut и After Effects. 
Наш проект состоит из:
1) css файлов;
   components.css
   layout.css
   base.css
2) html страниц;
   index.html
   AE.html
   CC.html
   contacts.html
   courses.html
   oplata.html
   examples.html
3) и изображений (логотипы и тд)

Используемые технологии - CSS3 и HTML5
Вся работа сделана в VSCode.
